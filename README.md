# <a id="home-page"></a> MyFavoriteAlbums Home Page

> MyFavoriteAlbums is a web application that analyzes ratings and rankings of music albums and graphically displays statistics of a dataset in many ways. Using a dataset, MyFavoriteAlbums is able to show the top albums in a range of years, top albums of a certain year, top artists, the albums of each artist, comparison between two artists, and top albums based on if the vinyl is not owned.
